# Hackathon Day Camp — Инструмент для преподавателей

Удобный веб-инструмент для организации и проведения игровых хакатонов. Позволяет быстро распределить учеников по командам, выбрать случайные условия (критерии) для проектов и следить за временем.

## 🚀 Как запустить проект

Проект состоит из статических файлов и простого Node.js сервера.

1. Убедитесь, что у вас установлен **Node.js**.
2. Откройте терминал в папке проекта.
3. Запустите сервер командой:
   ```bash
   node server.js
   ```
4. Откройте в браузере адрес: `http://localhost:3000` (если порт 3000 занят, сервер автоматически выберет следующий свободный порт).

## 🛠 Инструкция для преподавателя

Приложение разбито на несколько экранов, которые помогут вам провести хакатон от начала до конца.

### 1. Экран «Настройка» (Wizard)
Это первый этап подготовки. Он разбит на три шага:
*   **Шаг 1: Ученики.** Скопируйте и вставьте список имен учеников (по одному на строку) в текстовое поле и нажмите «Добавить». Вы увидите «плашки» с именами внизу.
*   **Шаг 2: Команды.** Выберите количество команд (2, 3 или 4). Вы можете распределить учеников вручную (перетаскивая их или кликая по именам) или нажать «Авто-распределение» для случайного деления.
*   **Шаг 3: Критерии.** Здесь вы можете настроить темы, которые будут выпадать в рулетке. Можно отключить ненужные категории или добавить свои варианты.

### 2. Экран «Рулетка»
После настройки команд перейдите в «Рулетку». Здесь для каждой команды выбираются условия их будущего проекта:
*   **Персонаж**, **Локация**, **Стиль**, **Палитра** и **Жанр**.
*   Вы можете нажать «КРУТИТЬ!» (все сразу) или выбрать режим «По одному».
*   **Важно:** Система старается выдавать уникальные результаты. Если одной команде выпал «Дракон», другой он, скорее всего, не выпадет, пока не закончатся остальные варианты.
*   Результаты сохраняются автоматически.

### 3. Экран «Команды» (Dashboard)
Это ваш командный центр во время работы учеников:
*   Здесь виден состав каждой команды и их выпавшие критерии.
*   Вы можете вписать **Приз**, который получит команда.
*   Нажмите кнопку **«Презентация»**, чтобы открыть красивое полноэкранное окно с результатами команды. Это удобно показывать на проекторе при защите проектов.

### 4. Экран «Таймер»
Встроенный таймер для контроля времени хакатона:
*   Удобные кнопки добавления времени (+1, +5 минут).
*   При приближении к концу время подсвечивается красным и начинает мигать.
*   Мини-таймер всегда виден в правом верхнем углу навигационной панели.

## 💾 Сохранение данных
Все данные (списки учеников, составы команд и результаты рулетки) сохраняются в памяти вашего браузера (**localStorage**). 
*   Вы можете смело перезагружать страницу — ничего не пропадет.
*   Чтобы начать новый хакатон с нуля, используйте кнопку **«Сброс»** на экране настройки.

## 🎨 Особенности интерфейса
*   **Адаптивность:** Плашки с именами учеников автоматически подстраиваются под размер экрана.
*   **Стиль:** Проект выполнен в ярком, дружелюбном стиле с плавными анимациями и понятными иконками.
*   **Кнопки:** Все важные действия выделены цветом, а кнопки имеют одинаковую высоту для удобства работы.
